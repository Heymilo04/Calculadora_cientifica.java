## DivisionPorCeroException

La excepción se lanza cuando el usuario intenta dividir entre cero.

Ejemplo:

```
if (b == 0) {
    System.out.println("No puedes dividir por cero");
}
------------------------------------------------------
Ejecución del programa

Ingrese el primer número: 4
Ingrese el segundo número: 0
la division por cero no es posible.
Resultado: Infinity
```
